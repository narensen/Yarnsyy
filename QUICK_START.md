## **Option 1: Use Batch Files (Easiest - Windows)**

1. **Double-click `START_BACKEND.bat`** - This starts the backend server
2. **Double-click `START_FRONTEND.bat`** (in a new window) - This starts the frontend server
3. **Open browser:** http://localhost:3000

---

## **Option 2: Manual Commands**

### **Terminal 1 - Backend:**
```powershell
python app.py
```
Wait for: `Running on http://127.0.0.1:5000`

### **Terminal 2 - Frontend:**
```powershell
npm run dev
```
Wait for: `Local: http://localhost:3000/`

### **Browser:**
Open: **http://localhost:3000**
